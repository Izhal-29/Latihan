Buatlah sebuah function untuk mengkoversi tahun menjadi abad / century contohnya seperti berikut tahun 1700 adalah abad 17, atau tahun 2010 adalah abad 21 Dengan keterangan sebagai berikut :

  • fungsi tersebut hanya bisa menerima tipe data number
  • kembalian nilai juga harus berupa number
contoh pengujian :
  1. 2010 // output 21
  2. 1699 // output 17
  3. 1700 // output 17
  4. 1 // output 1
  5. 100 // output 2
