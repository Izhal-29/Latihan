Soal 3

buatlah sebuah fungsi untuk mengubah time conversion dari 12 jam menjadi 24 jam. dengan keterangan sebagai berikut :

  • fungsi tersebut hanya bisa menerima tipe data string
  • kembalian nilai juga harus berupa string
contoh pengujian :
  1. 07:05:45PM // 19:05:45
  2. 02:05:45AM // 02:05:45
  3. 12:45:00AM // 00:45:00

Masukkan time dengan format jam:menit:detikAM/PM
contoh: 01:03:00PM
