Soal 4

Buatlah sebuah fungsi yang akan membalik urutan kata dari sebuah kalimat.
rules :

  • fungsi hanya bisa memproses string
  • jika string hanya terdiri dari 1 kata, tampilkan kata tersebut
contoh pengujian :
  1. reverseWords("Belajar javascript itu menyenangkan sekali") // "sekali menyenangkan itu javascript Belajar"
  2. reverseWords("Eduwork") // "Eduwork"
  3. reverseWords("Muhamad Aziz") // "Aziz Muhamad"

Masukkan kata atau kalimat
