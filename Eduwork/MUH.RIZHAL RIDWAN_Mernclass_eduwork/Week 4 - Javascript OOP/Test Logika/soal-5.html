Soal 5

Buatlah sebuah fungsi yang akan menghapus karakter pertama dan karakter terakhir pada sebuah string
rules :

  • Fungsi hanya bisa memproses string
  • jika string kurang dari 3 karakter, maka kembalikan nilai false
contoh pengujian :
  1. deleteChars("Belajar javascript itu menyenangkan sekali") // "elajar javascript itu menyenangkan sekal"
  2. deleteChars("Eduwork") // "duwor"
  3. deleteChars("yo") // false

Masukkan kata atau kalimat
