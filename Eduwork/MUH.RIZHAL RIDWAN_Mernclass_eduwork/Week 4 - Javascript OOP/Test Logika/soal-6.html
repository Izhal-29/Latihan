Soal 6

Buatlah sebuah fungsi yang akan mengembalikan jumlah total dari element array yang bernilai positif
rules :

  • fungsi hanya bisa memproses input berupa array of number
contoh pengujian :
  1. sumPositive([1,-4,7,12]) // 20
  2. sumPositive([-1,-4,-7,-12]) // 0
  3. sumPositive([-1,-4,"7",-12]) // false

Masukkan angka berapapun dipisah dengan koma, string menggunakan (" ")
